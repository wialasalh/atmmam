# الخطة المتكاملة — أتمم

## الرؤية
منصة متكاملة تدمج الباقات، الاشتراكات، التذاكر، المستندات، والموظفين في نظام واحد متماسك. كل عميل له منشآته، وكل منشأة لها باقة نشطة تحدد حدود الخدمة.

---

## 1️⃣ admin/packages — إدارة الباقات
**الهدف**: صفحة كاملة لإدارة الباقات (إضافة/تعديل/حذف/تفعيل) مع عرض احترافي.
- بطاقات عرض محسّنة مع Popular badge + زر تعديل تحت كل بطاقة
- مودال إضافة/تعديل مع مزايا تفاعلية (نقاط قابلة للإضافة/الحذف)
- تفعيل/إيقاف + حذف مع تأكيد + بحث ذكي
- كل باقه تحدد: عدد الموظفين المسموح، سعر الموظف الإضافي، الضريبة
- **الحالة**: موجودة أساساً — تحتاج إعادة بناء وتحسين

## 2️⃣ 🔗 تكامل التذاكر مع الاشتراكات
**الهدف**: فريق الدعم يعرف فوراً وضع العميل الاشتراكي عند فتح تذكرته.
- كرت "الباقة النشطة" في لوحة الأدمن تحت بيانات المنشأة:
  - اسم الباقة + السعر + الحالة (ساري/منتهي)
  - تاريخ البداية والنهاية + عدد الموظفين المشمولين
  - شريط تقدم الاشتراك
- عند إنشاء التذكرة من العميل: يعرض الباقة النشطة للمنشأة المحددة
- API: `/api/client/active-subscription?client_id=X`
- إضافة `subscription_id` في جدول التذاكر (اختياري — ممكن lookup ديناميكي)

## 3️⃣ 🏢 ممثل المنشأة (جديد)
**الهدف**: كل منشأة لها ممثلين/موظفين مسجلين.
- جدول `client_employees`:
  - id, client_id, full_name, phone, email, position, is_active, created_at
- تبويب "الموظفين" داخل صفحة `/dashboard/companies`
- إضافة/تعديل/حذف موظفين
- صلاحية الممثل: رفع مستندات + إنشاء تذاكر نيابة عن المنشأة
- ربط بعدد الموظفين المسموح به في الباقة (max_employees)

## 4️⃣ 📄 مستندات المنشأة (تطوير)
**الهدف**: كل منشأة لها مستودع مستندات منظم.
- كل منشأة لها مستنداتها الخاصة (موجود جزئياً في companies page)
- تصنيف: أساسية / تشغيلية / قانونية / عقود
- رفع متعدد + سحب وإفلات (موجود جزئياً)
- تواريخ صلاحية + تنبيهات انتهاء (موجود في Dashboard الرئيسي)
- عرض المستندات حسب المنشأة في لوحة الأدمن
- تكامل مع التذاكر: المستندات تظهر في شريط التذكرة الجانبي

## 5️⃣ 🔑 تغيير كلمة المرور
**الهدف**: العميل يغير كلمة سرّه من لوحة التحكم.
- تبويب/قسم في `/dashboard/profile`
- كلمة حالية ← كلمة جديدة ← تأكيد
- توثيق عبر Supabase Auth (updateUser)

## 6️⃣ 📅 تفعيل الاستشارة
**الهدف**: زر "جدولة استشارة" يفتح فورم مخصص.
- `?type=consultation` في رابط التذكرة الجديدة
- فورم استشارة: الموعد، وسيلة التواصل (عن بعد/حضوري)، الموضوع
- تظهر كـ "استشارة" في قائمة التذاكر عند الأدمن
- الفلترة: التذاكر العادية vs الاستشارات

## 7️⃣ ⚡ تحسينات إضافية
- مقارنة الباقات جنباً إلى جنب (جدول مقارنة)
- تحسين sidebar العميل بروابط مختصرة + أيقونات أفضل
- Search و Filter متقدم في admin packages
- أزرار عائمة في لوحة العميل (موجود): "الإبلاغ عن مشكلة" + "جدولة استشارة"
