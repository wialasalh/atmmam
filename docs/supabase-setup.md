# تشغيل قاعدة بيانات لوحة أتمم

1. أنشئ مشروع Supabase في منطقة قريبة من السعودية.
2. طبّق ملفات `supabase/migrations` بالترتيب.
3. انسخ `.env.example` إلى `.env.local` وأضف URL وAnon Key. مفتاح Service Role مخصص للخادم ولا يُعرض في الواجهة.
4. أنشئ المستخدم الإداري الأول من Supabase Auth، ثم غيّر دوره في `profiles` إلى `admin` من SQL Editor.
5. أعد تشغيل خادم Next.js. عند وجود المفاتيح سيصبح كل مسار تحت `/admin` محميًا تلقائيًا.

قبل إدخال بيانات حقيقية، اختبر حسابات الأدوار الأربعة وتأكد أن موظف العمليات لا يستطيع إدارة المستخدمين أو حذف المستندات.
